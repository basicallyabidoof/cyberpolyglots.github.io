# Cyberpolyglots

Welcome to the official GitHub repository for the **Cyberpolyglots** Village website.

This repository hosts the static website for our Hacker Conference Village, aimed at unifying polyglots in cybersecurity.

## Live Website
[https://cyberpolyglots.org](https://cyberpolyglots.org)

## About
Cyberpolyglots is a community dedicated to bridging the gap between spoken languages and cybersecurity. Our mission is to unify multilingual professionals and enthusiasts, fostering global collaboration and knowledge sharing in the infosec space.

## Development
This is a simple static site built with HTML and CSS.
Modifications to the `index.html` and `style.css` in the `main` branch are automatically deployed via GitHub Pages.
